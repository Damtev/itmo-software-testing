# Frontend

Frontend runs on localhost:8080

**Dependencies installation:**

`npm install`

**Run:**

`npm run serve`

**Unit tests:**

`npm run test:unit`

**e2e tests:**

`npm run test:e2e`

# NodeJS Backend

Listening port 3000

**Dependencies installation:**

`npm install`

**Run:**

`npm start`

# Spring backend

Listening port 3000

**Dependencies installation:**

`mvn install`

**Run:**

`mvn spring-boot:run`
